# Personal website content management system

> this is web CMS, the backend api service is [here](https://github.com/Jin-Yanhong/blogServer).

## Technology stack

-   vue3
-   typescript
-   element plus
-   paina
-   vue router
-   echarts

## How to start

firstly run

```
npm install
```

and then

```
npm run serve
```
